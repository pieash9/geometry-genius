DOM full form is Document Object Model. It's a programming interface for HTML and XML documents, which represents the structure of web page like tree where each elements are represented as node in the tree.
The DOM's function is to give programs tools access to and dynamic control over a web page's content and structure. Developers can change the appearance and layout of HTML elements, add, remove, and modify HTML elements. DOM allows developers to create interactive web pages that can respond to user actions, update their content dynamically, and provide a better user experience.

I will select HTML elements using DOM by using these method:

  1. 1. getElementById() : It select an element through element specified id.
  2. 2. getElementsByClassName() : It select an element through element specified class.
  3. 3. getElementsByTagName() : It select an element through element specified tag.
  4. 4. querySelector() : This method returns the first element that matches the specified CSS selector.
  5. 5. querySelectorAll() : This method returns a collection of all elements that match the specified CSS selector.

When an event, like a button click or key press, is triggered on an element, the target element handles it first, then its parent element, and so on, until the event reaches the top-level parent element (typically the document object), or until an event listener explicitly stops the propagation of the event.

A callback function is a function that receives an argument from another function and is subsequently called from inside the outer function to finish a task.
Callbacks are commonly used in JavaScript to handle events, process data from APIs or databases, and perform other tasks that require time-consuming operations.
